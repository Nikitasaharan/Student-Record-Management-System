# Student-Record-Management-System

## Objective
Implement a Student Record Management System using:
- File Handling
- Java Collections Framework
- Comparator, Iterator, RandomAccessFile
- Persistent Storage (students.txt)

---

## Folder Structure
- src/
- ├── model/Student.java
- ├── util/FileUtil.java
- ├── service/StudentManager.java
- └── Main.java

---

## Features
- Load records from file at program start  
- Save records back to file on exit  
- Add, search, delete & view students  
- Sort students by marks (Comparator)  
- Display using Iterator  
- Uses ArrayList + File Handling
